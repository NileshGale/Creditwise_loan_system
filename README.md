# CreditWise Loan System

## Overview

CreditWise Loan System is a machine learning project developed to predict loan approval status based on applicant information. The system analyzes various customer attributes and helps determine whether a loan application is likely to be approved.

## Objectives

* Perform data preprocessing and cleaning.
* Analyze applicant data through exploratory data analysis (EDA).
* Train and evaluate machine learning models.
* Predict loan approval outcomes accurately.

## Features

* Data preprocessing and feature engineering.
* Exploratory Data Analysis (EDA).
* Data visualization for insights.
* Machine learning model training and evaluation.
* Loan approval prediction.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook
